Documentação da API

Como integrar e usar os tokens de IA em suas aplicações

Para autenticar suas requisições à API de IA, inclua seu token no cabeçalho Authorization:

curl -X POST \
  https://api.dpedrosolucoes.com/ai/v1/chat/completions \
  -H 'Authorization: Bearer SEU_TOKEN_AQUI' \
  -H 'Content-Type: application/json' \
  -d '{
    "model": "gpt-4",
    "messages": [{"role": "user", "content": "Olá, como você está?"}]
  }'

Completions de Chat

POST /ai/v1/chat/completions

Cria uma completion de chat para a mensagem fornecida.

Geração de Imagens

POST /ai/v1/images/generations

Cria uma imagem com base em um prompt textual.

Moderação de Conteúdo

POST /ai/v1/moderations

Classifica se o texto viola as políticas de conteúdo.

Seus tokens de API estão sujeitos aos seguintes limites:

  • Requisições por minuto: 60 requisições
  • Tokens por dia: Variável conforme seu plano
  • Tamanho máximo de requisição: 4MB
  • Tempo limite de requisição: 30 segundos

Você pode verificar seu uso atual a qualquer momento através desta interface.
